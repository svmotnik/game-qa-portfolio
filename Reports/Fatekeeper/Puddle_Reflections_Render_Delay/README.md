__ID__: QA-0001

__SUMMARY__: Reflections in puddles don't render right away after exiting inventory.

__SEVERITY__: Minor

__PLATFORM__: PC

__BUILD__: 0.1.2

__ENVIRONMENT__:
- i7-8700
- NVIDIA RTX 3060 Gainward Ghost
- 24GB RAM HyperX Fury 2666Hz
- SSD
- Graphics set to max with DLSS on Quality

__EXPECTED__: Reflections are rendered right away after exiting inventory.

__ACTUAL__: Reflections have slight delay when rendering.

__STEPS__:
1. Enter area with puddles.
2. Aim camera towards the puddles.
3. Enter inventory using TAB.
4. Exit inventory using TAB.
5. Watch as the reflections on the puddles starts to turn on.
