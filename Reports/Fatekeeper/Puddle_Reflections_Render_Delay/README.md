ID: QA-0001

SUMMARY: Reflections in puddles don't render right away after exiting inventory.

SEVERITY: Minor

PLATFORM: PC

BUILD: 0.1.2

ENVIRONMENT:
- i7-8700
- NVIDIA RTX 3060 Gainward Ghost
- 24GB RAM HyperX Fury 2666Hz
- SSD
- Graphics set to max with DLSS on Quality

EXPECTED: Reflections are rendered right away after exiting inventory.

ACTUAL: Reflections have slight delay when rendering.

STEPS:
1. Enter area with puddles.
2. Aim camera towards the puddles.
3. Enter inventory using TAB.
4. Exit inventory using TAB.
5. Watch as the reflections on the puddles starts to turn on.
